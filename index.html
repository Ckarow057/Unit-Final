Exposing Localhost with Ngrok

Ngrok is a tool that allows developers expose a url or local server to the public. This is used for:

  • Testing webhooks
  • Sharing work in progress
  • Remote access to local servers
  • Free web hosting

Key Reasons to use Ngrok

  • Easy to set up and use
  • Secure tunnels with HTTPS
  • Inspect traffic and replay requests

Steps to Use Ngrok

  1. Download and install Ngrok
    • Visit the Ngrok download page
    • For Mac, use brew install ngrok
    • For Windows, download the zip file and extract it
  2. Sign up for a free account to get an authentication token
  3. Connect your account using the command: 
    ngrok authtoken YOUR_AUTH_TOKEN
  4. In the terminal, run the command: 
    ngrok http http://localhost:3000
  5. Ngrok is now exposing the backend or url hosted on localhost:3000
  6. Ngrok will provide a public URL (e.g., https://abcd1234.ngrok.io) that tunnels to your local server
  7. Follow the link in the console to a web browser to test your api or view your webpage
